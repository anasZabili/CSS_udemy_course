Attribut selecteur:
  les selecteur d'attribut servent a selectionner des elements
  qui possede certain attribut html

  - h2[class=subtitle]: selection tout les h2 possedant l'attribut class
    égale a subtitle
  - img[src^="../"]: selection tout les image dont la source commence par la
    chaine de caractere "../" ce patern matching est fait grace a la presence de
    l'élément ^
  -img[src$=".jpg"]: selection tout les image dont la source termine par la
    chaine de caractere ".jpg" ce patern matching est fait grace a la presence de
    l'élément $
  -img[src*=".jpg"]: selection tout les image dont la source contient la
    chaine de caractere ".jpg" ce patern matching est fait grace a la presence de
    l'élément *
  -h2[class~="article-subtitle"]: ~ est le withspace selector, il selection tout
    les article-subtitle separer (ou non) par l'espace

    voir auss |="xxxx" selection avec le tiret

Hex Code explication:
  #rrggbb

  chaque degit allant de 0 à f, correspond a une intensité de couleur primaire

Background: 
  -background: url("xxx") = set une background image 
  -background-repeat: no-repeat = empecher la repetition de limage sans combler
    le container
  -background-size: heigthpx widthpx; = specifie la taille de notre background
    image en resizan l'image (souvent une mauvaise pratique qui deforme l'image) 
  -background-size: cover; = force notre image a prendre toute la taille de son 
    container en denaturant les proportion de limage
  -background-size: contain; = resize notre image sans denaturer les proportion
    pour qu'il occupe au mieux lespace de son container

  Uniter de taille:
    Absolue:
     px: taille standard relative sadapte a la resolution de lecran

    Relative: 
      vw: View Widht est une unité relative 100vw = 100% de la largeur de lecran
        disponnible, pas la largeur du parent mais de lecran
      vh: View Height est une unité relative 100vh = 100% de la hauteur de lecran

      em: 1 em = une taille de font size standard (unité absolue), ce base sur la taille standard
        des lettres, ainsi 2em double la taille d'un taexte, tres cool car
        cette unité s'adapte au different media telephone, ordi etc, pour une taille
        de texte on privilegie TOUJOURS cette unité plutot que px 
  
  Text and Font:
    Le texte traite tout ce qui a un rapport seulement avec le text (color etc)
    La font traite ce qui a un rapport avec la font (size etc)

    text-decoration: underline/line-through/overline
    text-transform: uppercase/lowercase/capitalize (tres interessant met juste 
      la premiere lette en majuscule)
    text-align: gere l'alignement du text au sein de son container
      justify/center/left/right

    font-size: 1.15em;
    font-weight: bold/[100-900];
    font-style: italic;

    Font-family: precise la type de police (serif, sans-serif, monospace) ou 
      le nom de la famille (calibri, Time Neww Roman)
      Pour set une font familly les schema a suivre :
          font-family: "First Font", seconde font;
          font-family: "Calibri", sans-serif;
          ici on specifie une font a calibri et si calibri nest pas disponnible
          alors on set a sans-serif, toujours specifier une font standard a tout 
          les navigateur comme derniere alternative car si, on specifie une font, 
          qui nest pas disponnible si un navigateur on souhaite toujours pouvoir,
          controller le style de la font

      importer une font: on selectionne un font sur https://fonts.google.com/, puis 
      une fois notre fotn chosii on cherche a mettre le moins de variant possibles
      car plus on aura de variant de font (ex bold 800, bold 900), plus le 
      chargement de notre font par l'application sera lente

FlexBox:

  Pour creer un flex container on chercher lelement parent de notre flex zone
  puis on applique la props display: flex (exactement comme un <Grid container>)
  
  flex-direction: row/column = indique a notre container la direction dans laquel
    il va render les item qu'il contient row et ligne de gauche a droite
    (ex A B C D),
    column = colonne de haut en bas 
    (ex: 
      A
      B
      C
      D
    )
    row-reverse: ligne de droite a gauche (ex: D C B A)
    column-reverse: colonne de bas en haut
    (ex: 
      D 
      C 
      B 
      A
    )
  
  flex-wrap: wrap/wrap-reverse/nowrap = flex-wrap: wrap, permet de specifier, le 
    comportement a adapter en cas d'overflow delement si "wrap" est specifier, 
    le flex container renvoie a la ligne suivante les element overflow, 
    "nowrap" par default entasse simplement les composant
    "nowrap" = affichage sur une seul lignes
    "wrap" = affichage sur plusieur lignes

  justify-content: flex-start/flex-end/center = aligne (horizontalement) les item dans un 
    container display flex
    space-between: remplie tout le container en mettabt de lespace entre les element
    space-arround: remplie tout le container en mettabt de lespace autour des element

  align-items: center/flex-start/flex-end: aligne verticalement
    stretch: valeur par def&ault etire les item sur toute le hauteur
    baseline: aligne sur la meme ligne tout les items

  order: {order} (la valeur par default est 0), order est une props importante permettant de specifier la 
    position d'un element sans avoir a modifier sa position dans le html,
    un element order:1 apparaitera avant un element order: 2, si collision le 
    premeier element de l'html est pris en compte 
    cette propriete prend encore plus de sens si le style est modifier avec du 
    javascript, cela va permettre de boyger les element sans modifier le htmlx

  flex-basis: xxpx defini la taille minimal d'un item dans un flex container 
  
  flex-grow: {priority} (la valeur par default est 0) props importante, elle permet a chaque flex item, 
    de prendre une certain proportion de lespace restant disponnible, 
    ainsi si apres avoir rendu tout les item si le container n'est pas rempli 
    lespace suplement sera distribué au item proportionnelement a leur priorité
    un flew-grow: 2 aura 2 fois plus dextra quun flex-grow: 1, il est necessaire
    de toujours definir cette propriete sur toutj les elemtn du container
    car dans le cas contraire, si des element nont pas de flex grow il ne prendrons
    aucun extra space, et seulement les seul element  prendrons de lextra space

  flex-shrink: {priority } (la valeur par default est 0), props tout aussi importante
     que flex-gorw, elle defini l'espace pris a chaque item lors d'un retreciseement
     ex un element de priority 1 perdra 2 fois moins despace lors d'un retrecissement
     qu'un element de prio 2 un element de prio "0" ne retrecie jamais
     astuce: un element de prio X ce reduit X fois plus vite qu'un element de prio 1

  flex: [flex-grow] [flex-shrink] [flex-basis] = cette priopriete est un 
    shortcut, pour deifni les 3 propriete flex

  align-self: center/flex-start/flex-end est un props a appliquer directemenet sur un item recouvrant
    la props align item de son container, elle permet d'appliquer un alignement 
    vertical particulier a un item sans prendre en compte l'alignement imposé
    par le flex container

  Animation:
    
    Transition:
      transition: [property] [duration (s | ms)] [transition style] [delay (for the animation to start)]
      exemple :     transition: background, padding 0.3s linear 0.1s;

        ici je fait en sorte de poser une transition sur le changeement du background 
        colro apres un hover en adoptant le style ease in out

    Animation:
      une animation ressemble a une transition mais c'est une fonction reutilisable
      ainsi pour faire une transition du type changer le background color, 
      de bleu a rouge, de 5 element, les transition devrons etre deifni sur les 
      5element, la ou les animation permet de definir une seul animation,
      appliqué sur 5 elements, de plus une animation peut être declanché, 
      au bout d'un delemnt au montage de la page sans attendre devent particulier

      creer une animation:
        une animation est composer de au moins deux "key-frame" une start et 
        une end
        @keyframes red-to-black {
          from {
            background: red;
          }
          to {
            background: black;
          }
        }
        ici une basique animation allant du rouge au noir est creer

        Creation danimation a plus de 2 etat de transition

        @keyframes red-to-black-to-blue {
          0% {
            background: red;
            transform: translate(0px, 0px);
          }
          50% {
            background: black;
            transform: translate(10px, 10px);
          }
          100% {
            background: blue;
            transform: translate(20px, 20px);
          }
        }
        ici on defini nos changement en fonction du pourcentage d'avancement de notre 
        animation, 

        Ajout de notre animation :
          Pour ajouter notre animation on selectionne lelement concerné auquel on ajout 
          c'est porps
            animation-name: red-to-black-to-blue;
              = doit être le nom exacte de notre keyframes
            animation-duration: 4s;
              = le temps que notre animation doit duré (par default 0)
            animation-timing-function: linear;
              = le style de course que lanimation adopte (linear, ease, ease-in, ease-out etc)
            animation-delay: 0.2s;
              = le temps que l'animation prend avant de start (peut être une value negatif)
            animation-iteration-count: infinite;
              = le nombre de fois que notre animation sera joué, (in chiffre ou infinite)
            animation-direction: alternate;
              = le facon dont le differente etape de notre animation seront executé
                alternate: une boucle est fait de 0% a 100% a 0% etc...
                alternate-reverse: une boucle inversé est faite de 0% a 100% a 0% etc..
                normal: single time de 0% a 100%
                reverse: single time de 100% a 0%

            animation: name duration timing-function delay iteration-count direction fill-mode;
              est un shortcut qui prend en compte tout les propriete
